# Beamer Presentation Project

This project contains a Beamer (LaTeX) presentation: `Presentazione.tex`

## Beamer Documentation

Local Beamer docs are available in `beamer_docs/markdown/`. **Do NOT read all files at once** — only read the specific doc relevant to the current question. Use the file names to pick the right one:

- `Creating-Frames.md` — frame environments, options, breaking frames
- `Creating-Overlays.md` — overlays, \pause, \only, \visible, \uncover
- `Structuring-Presentation-The-Static-Global-Structure.md` — sections, parts, table of contents
- `Structuring-Presentation-The-Interactive-Global-Structure.md` — navigation, buttons
- `Structuring-Presentation-The-Local-Structure.md` — blocks, columns, lists, theorems
- `Themes.md` — built-in themes overview
- `Inner-Themes-Outer-Themes-Templates.md` — customizing inner/outer themes and templates
- `Colors.md` — color themes, setbeamercolor, color palettes
- `Fonts.md` — font themes, setbeamerfont
- `Graphics.md` — including images
- `Animations-Sounds-Slide-Transitions.md` — animations and transitions
- `Adding-Notes-Yourself.md` — speaker notes
- `Creating-Handouts-Lecture-Notes.md` — handout mode
- `How-To-Uncover-Things-Piecewise.md` — progressive reveal techniques
- `Guidelines-Creating-Presentations.md` — best practices

## Build

Compile with `pdflatex` via LaTeX Workshop (Ctrl+Alt+B).
